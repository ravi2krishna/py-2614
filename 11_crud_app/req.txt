-> Create Student Management System which helps Admins of an Institute To Manage Students Info 

-> Student has Information like IDs, Names, Scores, SKills etc 

-> System Setup 

    -> Company Name: Digital Tech 
    -> Software Name: Student Management System 
    -> Version: v1 

    -> Above Data should be READ ONLY


-> Admin Info 

    -> Admin Contact Number: 9999999999
    -> Admin Email ID: admin@digital.com 

    -> Above Data should be READ ONLY

-> When this Application is Started, Display System Info 

-> Core Functionalities (CRUD)

    -> Add Student (CRUD - C - CREATE)

        -> Prompt for Student ID and check if this id already exists in the system 

            -> If ID exists, say Student ID already Exists 

            -> If ID Doesn't exists, Add the Student ID in the system 

                -> Take Student Name 

                    -> Format Name like below 

                        -> User input (ravi krishna) => Output (Ravi Krishna) store in system 

                -> Take Student Scores 

                    -> Take Scores till the Student says done 

                        -> Enter Score or type done 
                        -> Validate Score between 0 to 100 
                        -> Scores can be Duplicated 

                -> Take Student Skills 

                    -> Take Skills till the Student says done 

                        -> Enter Skill or type done 
                        -> Skill cannot be Duplicated (Unique Skills Only)

    
    -> Modify/Edit Student (CRUD - U - UPDATE)

        -> Search By ID 
        -> If ID present, allow to update only Name 
        -> If ID not present, give appropriate Error Message 

    -> Delete Student (CRUD - D - DELETE)

        -> Remove By ID 
        -> If ID present, Delete
        -> If ID not present, give appropriate Error Message 

    -> Read Student Info (CRUD - R - READ)

        -> Display By ID 

        -> If ID present, Show Following Details

            -> ID 
            -> Name
            -> All Scores
            -> Average Score 
            -> Highest Score 
            -> Lowest Score 
            -> All Skills 
            -> Skills Count 

        -> If ID not present, give appropriate Error Message  


-> When this Application is Closed, Display Admin Info
