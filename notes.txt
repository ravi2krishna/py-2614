        -> An Identifier is a "name" used to uniquely identify piece of data in code 

            -> Variable 
            -> Function 
            -> Class  
            -> Object
            -> Method
            -> etc 

        -> Identifier Rules 

            -> Can Contain 

                -> Alphabets (a-z A-Z)
                -> Digits (0-9)
                -> Special Symbols (Only _ (underscore) is allowed)

            
            -> Must Start With 

                -> An Alphabet or _ (underscore)

                -> NOTE: Replace space with underscore for long words 

        -> NOTE: Always use proper way of naming the identifiers  

        -> NOTE: Always use UPPER CASE for Static Data (Fixed / Doesn't Change)



    -> Variables 

        -> Variable is a named storage location in Computers Memory (RAM)
        that holds some Value or Data.

        -> Syntax 

            -> variable_name = value 

                -> variable_name should follow identifier rules 
                -> = Assignment Operator 
                -> value is data that you want to store 

                -> data = 10 (valid)
                -> data = ten (invalid)
                -> data = "ten" (valid)

        -> Variables Allow You To 

            -> Assign Data (Store Data)

            -> Retrieve Data (Get Data)

            -> Manipulate Data (Change Data)
